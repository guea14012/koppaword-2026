Professional editing tools wrapped in a cyberpunk interface

✍️

Rich Text Engine

TipTap v2 powered editor with headings, tables, code blocks, task lists, images, links, and full typography control.

🤖

AI Writing Assistant

Powered by Claude. Rewrite, summarize, translate, fix grammar, expand ideas — directly in your document.

⌨️

Command Palette

30+ commands, fuzzy search, keyboard navigation. Press Ctrl+K to launch instantly.

🗂️

Multi-tab Editing

Open multiple documents simultaneously. Each tab is independent with its own undo history.

💾

Auto-save + History

Automatic saves every 30 seconds. Version history keeps up to 50 snapshots per document in local SQLite.

📁

File Format Support

Native .kwdoc format, import .docx and .txt, export to HTML. Double-click .kwdoc files to open instantly.

Futuristic UI

Glassmorphism panels, neon accents, animated ribbon toolbar, Focus Mode, Presentation Mode. No subscription required.

🔒

Works Offline

All editing, saving, and history features work completely offline. AI requires an Anthropic API key (optional).

💻

Cross-Platform

Windows, Linux, macOS. Built on Electron 29 with native OS integration — file associations, shortcuts, menus.
